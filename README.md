# Prototipo CENATRA - Tiempo de espera para trasplantes

Aplicación web desarrollada en Streamlit para estimar el tiempo de espera de un trasplante en función de variables demográficas y administrativas.

## Modelos incluidos
- OLS
- RLM (HuberT)

## Dataset
Datos públicos del CENATRA, primer trimestre de 2025.
